# 🚀 JavaScript Journey: From "Silly" to Solid

Welcome to my JavaScript learning playground! This repository is a collection of my daily practices, logic experiments, and mini-projects as I revise and master JS. 

Every folder here represents a "lightbulb moment" or a practice session to sharpen my frontend skills.

## 📁 What's Inside?
Currently, this repo houses various small-scale applications including:
* **User Greeting:** Basic DOM manipulation using inputs and labels.
* **Modern Counter:** A functional counter app with increment, decrement, and reset logic.
* **Data Types & Conversions:** Experiments with typecasting and arithmetic operations.
* **UI/UX Tweaks:** Applying modern CSS (Flexbox, Dark Mode) to simple logic programs.

## 🛠️ Tech Stack
* **HTML5** - Structuring the web pages.
* **CSS3** - Making things look modern and responsive.
* **JavaScript (ES6+)** - Handling logic, DOM events, and functionality.
